# 体育场馆

体育场馆预约系统：`https://ggtypt.nju.edu.cn/venue/home`。

先完成统一认证登录：

```bash
nju-cli login
```

## 场馆搜索

按校区列出可预约项目：

```bash
nju-cli venue list
```

示例输出：

```
## 仙林校区
- 乒乓球（方肇周体育馆）
- 信息中心北篮球场（室外场地）
- 健身房跑步机（四组团体育馆）

## 浦口校区
- 健身房跑步机（左涤江运动馆）
- 健身房（左涤江运动馆）
- 兵乓球（左涤江运动馆）
```

按条件过滤：

```bash
nju-cli venue list --campus 仙林 --sport 跑步
```

查看某个预约项目详情：

```bash
nju-cli venue site 健身房跑步机 --campus 仙林 --venue 方肇周
```

示例输出：

```
## 方肇周体育馆 健身房跑步机
校区：仙林校区
开放时间：09:00-22:00
可预约场地数：2
联系电话：89680100
```

## 预约

日常预约直接用 `book`，它会自动完成状态检查、订单确认、验证码获取、`86maid/ddddocr` 识别、验证码校验、提交预约和免费付款确认：

```bash
nju-cli venue book \
  --site 健身房跑步机 \
  --campus 仙林 \
  --venue 方肇周 \
  --date 2026-06-07 \
  --space 6号跑步机 \
  --time 21:30
```

默认不保存验证码图片。需要调试 OCR 时可以指定保存位置：

```bash
nju-cli venue book \
  --site 健身房跑步机 \
  --campus 仙林 \
  --venue 方肇周 \
  --date 2026-06-07 \
  --space 6号跑步机 \
  --time 21:30 \
  --captcha-output /tmp/nju-venue-captcha.png
```

如果出现验证码识别失败，可以重新提交一遍（这样就会拿到新的验证码，就有可能成功了）。

如果只想提交、不自动发送免费付款确认请求，加 `--no-pay`。如果要查看完整中间结果，加 `--json`。

## 预约管理

列出预约记录：

```bash
nju-cli venue orders --size 20
```

查看预约详情：

```bash
nju-cli venue order 826433
```

取消预约：

```bash
nju-cli venue cancel D260607001596
```
